# AegisGovern_v1 Risk Scoring Methodology

## Risk Assessment Approach

Risk ratings were determined using a qualitative risk assessment model based on Likelihood and Impact.

Risk Score = Likelihood × Impact

---

## Likelihood Scale

| Score | Description    |
| ----- | -------------- |
| 1     | Rare           |
| 2     | Unlikely       |
| 3     | Possible       |
| 4     | Likely         |
| 5     | Almost Certain |

---

## Impact Scale

| Score | Description |
| ----- | ----------- |
| 1     | Negligible  |
| 2     | Low         |
| 3     | Moderate    |
| 4     | High        |
| 5     | Critical    |

---

## Risk Classification Matrix

| Score Range | Risk Level |
| ----------- | ---------- |
| 1–5         | Low        |
| 6–10        | Medium     |
| 11–15       | High       |
| 16–25       | Critical   |
